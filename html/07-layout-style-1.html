화면 배치(표시, 형식) 지정 속성 : display

    요소가 화면에 어떻게 표시될지 형식을 지정하는 속성 

    - block : 요소를 한 줄 전체를 차지하고 독립된 박스를 형성하게 만드는 속성
             + 높이와 너비 모두 지정 가능.

    - inline : 요소를 콘텐츠 크기만큼 차지하며, 같은 줄에 나란히 배치되도록 만드는 속성입니다.
               + 높이 지정 x, 내용에 따라 너비가 자동조정 

    - inline-block : 화면/ 영역을 열로 분할(inline)
                     + 크기 지정 o(block)

    - none : 요소를 화면에서 숨김 (실제로는 존재함)

    - flex : 부모 요소 안의 자식 요소들을 정렬, 배치할 수 있는 유연한 박스 모델을 제공.

    - grid : 부모 요소를 행(row)과 열(column) 기반의 격자(Grid) 레이아웃으로 설정, 자식 요소를 정렬하고 배치 가능.
