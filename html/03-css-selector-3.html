일반 구조 선택자(위치 기준)

    - 형제 관계에 있는 요소 중 특정 요소만 선택하는 선택자

    **** 중요 ****
    위치를 기준으로 구분!!
    
    [작성법]

    (A : 선택된 요소)
    A:first-child -> 형제들 중 A가 첫 번째로 위치한 요소가 맞을 경우 선택 

    A:last-child 
    -> 

    a:nth-child(n) 
    ->

    A:nth-last-child(n) 
    -> 형제들 중 A가 뒤에서 n번째 일 경우

형태 구조 선택자

    
      - 선택된 형제 관계 요소 중 특정 요소를 선택하는 선택자
      (선택된 요소들을 기준으로 선택)

      [작성법]
      :first-of-type : 선택된 형제 요소 중 첫번째만 선택
      :last-of-type : 선택된 형제 요소 중 마지막 선택
      :nth-of-type(n) : 선택된 형제 요소 중 n번째 선택
      :nth-last-of-type(n) : 선택된 형제 요소 중 뒤에서 n번째 선택

부정 선택자(:not)

    
      - 선택된 요소를 제외한 나머지를 선택

      [작성법]
      A:not(B)
      -> 선택된 A들 중에서 B를 뺀 나머지만 선택
